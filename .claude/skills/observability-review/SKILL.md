# Skill: Observability Review

Review observability of the implemented feature.

---

## Logging

Verify:

- Structured logs
- Useful messages
- No sensitive data
- Error logs

---

## Metrics

Check if important metrics exist.

Examples:

- Requests
- Errors
- Processing Time

---

## Tracing

Verify:

- Correlation ID
- Request ID
- Trace propagation

---

## Monitoring

Identify:

- Missing metrics
- Missing alerts
- Missing dashboards

---

## Error Visibility

Can production incidents be diagnosed easily?
