# Skill: Documentation Review

Review all documentation related to the implemented feature.

---

## README

Verify:

- Installation
- Running locally
- Environment variables

---

## API

Check:

- Swagger/OpenAPI
- Request examples
- Response examples

---

## Architecture

Verify:

- Diagrams
- Flow description
- Decisions explained

---

## Code Documentation

Evaluate:

- JavaDocs when necessary
- Comments
- Self-documenting code

---

## Feature Documentation

Check:

- Business rules
- Limitations
- Assumptions

---

## Developer Experience

Can another developer understand this feature in less than 10 minutes?

---

### Documentation Score

### Missing Documentation

### Improvements
