# Skill: Security Review

Review the feature as if preparing it for production.

---

## Authentication

- JWT
- Session
- OAuth
- Token expiration

---

## Authorization

Check:

- Role validation
- Resource ownership
- Endpoint protection

---

## Input Validation

Verify:

- Bean Validation
- Sanitization
- Path Variables
- Request Params

---

## Injection

Look for:

- SQL Injection
- JPQL Injection
- Command Injection

---

## Sensitive Data

Never expose:

- Passwords
- Tokens
- Secrets
- Internal Exceptions

---

## HTTP Security

Review:

- CORS
- Headers
- HTTPS assumptions

---

## File Upload

If applicable:

- File validation
- MIME validation
- Size limits

---

## Logging

Ensure logs never expose:

- Passwords
- JWT
- Personal information

---

## Dependencies

Highlight outdated or vulnerable dependencies if identified.

---
